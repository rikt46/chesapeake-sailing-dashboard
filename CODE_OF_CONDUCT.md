# Code of Conduct

## Our standard

This project welcomes contributions from anyone. Participants are expected to
be respectful and constructive in all interactions — issues, pull requests, and
any other project spaces.

## Scope

This standard applies to all project spaces: GitHub issues, pull requests,
discussions, and any other venue where someone is representing the project.

## Enforcement

Instances of unacceptable behavior may be reported to the maintainer via
GitHub's private messaging or the security advisory link in SECURITY.md.
All reports will be reviewed promptly. The maintainer reserves the right to
remove comments or contributions that do not meet this standard, and to ban
participants who repeatedly violate it.
